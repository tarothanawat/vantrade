# Software Architecture Project — Submission Criteria

## Functional Requirements

### 1. Multi-User Role System
- Must include **three distinct user roles**
- Each role must have:
  - Its own responsibilities
  - Its own permissions
  - Controlled access to system features

### 2. Full Database Operations
- Each user role must be able to perform **at least one CRUD operation**

### 3. Authentication & Authorization
- Secure login/logout
- User authentication
- Role-based authorization
- Access control based on permissions
